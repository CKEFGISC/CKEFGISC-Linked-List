## 編譯方式
將所在位置切換到專案根目錄底下：<br>
```
$ cd PATH_TO_FOLDER
```
並以 make 編譯：<br>
```
$ make         # 打包成函式庫
$ make example # 編譯 exmaple.cpp
```
之後便可以用以下指令來執行<br>
```
$ ./example
```
