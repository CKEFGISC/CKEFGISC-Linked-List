你只要把它用 template 改寫<br>
讓我們的 LinkedList 能存任何型別的變數就行了<br>
## 編譯方式
將所在位置切換到專案根目錄底下：<br>
```
$ cd PATH_TO_FOLDER
```
並以 g++ 編譯：<br>
```
$ g++ -o example example.cpp
```
之後便可以用以下指令來執行<br>
```
$ ./example
```
